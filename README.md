

# HungryHub
HungryHub: A full-stack MERN (MongoDB, Express, React, Node.js) food delivery web application. Features include user authentication, restaurant listings, cart management, and secure checkout. Designed for a seamless online food ordering experience.
## Features
- User authentication
- Restaurant listings
- Cart management
- Secure checkout
- Responsive design
